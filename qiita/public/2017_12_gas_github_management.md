# 背景

Google Apps Script(GAS)はウェブIDE上で完結するサーバレスにスクリプトを書けるサービスですが、ソースコードの管理ができない問題があります。(スナップショットの管理は可能みたいです)

# できること

IDEにGithub関連のメニューが追加され、基本的な操作を行うことができます。
