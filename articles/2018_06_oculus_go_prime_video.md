# よく紹介されている手順

1. Oculus Goのホームより、ナビゲーション > ブラウザを選択します
2. 検索窓で「Amazon Prime Video」のページを表示します
3. ログインします
4. **右上の「デスクトップをリクエスト」ボタンをクリックしてから再生ボタンを押します**
5. 動画が再生されます

# 自分の場合

上記の方法を試すと、「デジタル権利エラー」というメッセージが表示され、再生されませんでした。

「Oculus Go Prime 見れない」とかでググると、「デスクトップをリクエスト」を確認しろというページしか出なくて途方にくれていました。

しかし、有志のツイートを頼りに解決することができました。
ぜひ一度下記の手順も試してみてください！

# 解決方法

1. ブラウザの右のタブの設定より、「閲覧データのクリア」を選択
2. 「サイト設定」にチェックをいれ、「データを削除」をクリック
3. 任意のPrime Videoを再生
4. **動画再生前に画面下部にアクセスの許可を求めるメッセージが表示されるため、「許可」をクリック**
    - この手順ですが、選択する前に次の状態に進んでしまうと上記のエラーが発生してしまいます。
    - この手順がうまく行かなかった場合は1からやり直してみてください。
5. 動画が再生されます
