#　観点

- 無料か、どのくらいお金かかるか
- マークダウン記法(MD)が使えるか
- 導入実績があるか
    - セキュリティとか的な懸念
- テンションが上がるか

# 候補

- Google Docs
    - 無料
    - 検索とか閲覧がしづらい
    - 外部の人とも共有しやすい
- esa
    - 1user 500円/月
    - MD使える
    - おしゃれ
- Githubのwiki
    - 無料
    - issueでタスク管理とかもすればGithub上だけで完結させられそうな雰囲気は良い
    - リポジトリが複数あるプロジェクトだとどこにまとめるのが適切なんだろうか
- Kibela
    - 5userまで無料
    - 以降1user 540円/月
    - MD使える
    - Wiki(しっかりめの文章)とブログ(カジュアルな文章)を使い分けられる
- コンフル
    - レガシーみがある(偏見)
- Crowi
    - 無料 (というかOSS)
    - MD使える
    - デプロイしないといけないのが個人的にだるい
- Dropbox Paper
    - Dropboxの容量内なら無料
    - MDかける
- Qiita:team
    - 規模ごとに金額が違う
        - https://teams.qiita.com/pricing/
    - Qiitaの機能が大体使える
        - もちろんMD使える

何か良さげなのがあれば教えてください〜
